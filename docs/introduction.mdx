AnswerLoops is an AI-powered community support platform. It watches the channels where your users ask questions, searches your knowledge base for a matching answer, and replies automatically — so your team handles edge cases and complex issues instead of answering the same question for the fiftieth time.

## How it works

1. **Ingest** — A user posts a question in Discord, Slack, GitHub Discussions, Telegram, or via Email or your website widget
2. **Search** — AnswerLoops runs a semantic vector search against your knowledge base to find the best-matching content
3. **Deflect** — If the AI's confidence exceeds your threshold (default 0.8), it posts an answer directly in the thread or channel
4. **Escalate** — If confidence is too low, it pings your support role so a human can respond
5. **Learn** — Resolved tickets can be promoted back to the knowledge base to close the loop

## Key features

| Feature | Description |
| --- | --- |
| AI deflection | Automatically answers questions using your docs, with a configurable confidence threshold |
| Multi-channel ingest | Discord (text \+ forum channels), Slack, GitHub Issues & Discussions, Telegram, Email, Website widget |
| Knowledge base | URL crawl (Firecrawl), file upload (PDF/DOCX/MD/TXT/CSV), GitHub repo sync, or promote resolved tickets |
| Knowledge gaps | Surfaces questions the AI couldn't answer so you can fill content holes |
| Human escalation | Pings a support role when AI confidence is too low |
| CSAT scoring | Post-resolution rating prompts sent back through the same channel |
| Simulation mode | Dry-run AI against past tickets — no replies sent, zero writes |
| Ticket list | Unified inbox with source badges (Discord, GitHub, Slack, Telegram, Email) |
| Multi-provider AI | OpenAI, Anthropic, Google Gemini, Groq, Mistral, Ollama |
| Website widget | Embeddable script tag with lead capture |

## Ingest channels

| Channel | How it connects | What ingests |
| --- | --- | --- |
| Discord | 1-click OAuth (cloud) or bot token (self-host) | Text channel messages \+ forum thread posts \+ replies |
| Slack | 1-click OAuth or polling mode | Channel messages |
| GitHub | GitHub App webhook | Issues, issue comments, Discussions, discussion comments |
| Telegram | Webhook via bot token | Messages in configured chats |
| Email | Provider-agnostic inbound webhook | Any inbound email forwarded to the webhook URL |
| Website widget | Embeddable `<script>` tag | Widget chat sessions |

## Deployment options

**Cloud** — Sign up at [app.answerloops.com](https://app.answerloops.com), no infrastructure or bot setup required. Discord connects in one click via OAuth.

**Self-host** — Run on your own servers using Docker Compose. Full data ownership, bring your own AI provider keys, all features available.
